Supporting

The NIST Immunization Test Suite Edition 2 supports a broad range of testing in support of the Immunization Community, including transport, messaging (content), and functional.

  • CDC SOAP Validation
  • HL7 V2.5.1 Immunization Message Implementation Guide Release 1.5 (2018 Update)
  • EHR and IIS Functional Capabilities
  • ONC Health IT Certification
    • (f)(1) SVAP 2024
    • (f)(1) HTI-2 Revised
ASTP ONC Health IT Certification Start Certification Testing Start Certification Testing

The NIST Immunization Test Suite Edition 2 is a test tool used for ASTP ONC certification of Health IT Modules. The NIST capability used for ASTP ONC Certification can be accessed by navigating to the ASTP ONC Certification Test Plan via the HL7 Context-based option. The ASTP ONC Test Plan is used to certify Modules for the 170.315(f)(1) Transmission to Immunization Registries ASTP ONC criterion SVAP 2024 and HTI-2 Revised.
