<li>The Tester copies this SOAP connectivityTest request message envelope and pastes it into the Test Tool, which is functioning as the receiver system </li>

<li>The Test Tool generates a Validation Report listing the results of the validation</li>
<li>The Tester reviews the Validation Report and compares the results to the Objectives and Evaluation Criteria associated with the SOAPENV_1_ConnectivityTest_Request Test Case</li>
