This test evaluates the capabilities of a system to generate a message conforming to the SOAP 1.2 standard and CDC WSDL1.0 that may be used for transporting an HL7 V2 message to an IIS. The body of the SOAP message shall contain the element SubmitSingleMessage, as defined in the CDC WSDL 1.2 with arbitrary conforming values for all child elements (username, password, facilityID and hl7Message)
