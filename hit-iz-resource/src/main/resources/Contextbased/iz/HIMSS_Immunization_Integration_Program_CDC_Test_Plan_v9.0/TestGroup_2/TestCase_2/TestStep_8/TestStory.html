Description

The nurse administers the inactivated influenza vaccine:

- Documents all required information for the vaccine.





Comments

No Comments


Pre-condition

Order is placed for inactivated influenza vaccine.


Post-Condition

The inactivated influenza vaccinations is recorded in the EHR.



Test Objectives

Record Vaccine Administration: The EHR or other clinical software system records information about each vaccine administered. The EHR records this information as structured data elements, including, at a minimum: date administered, administering clinician, site of administration (e.g., left arm), immunization type,  lot number, manufacturer, Vaccine Information Statement date, quantity of vaccine/dose size and ordering clinician. The system also assures data quality, i.e., data entered are appropriate (e.g., avoid “oral” route for IM vaccines, and assures dose is appropriate for the vaccine).




Evaluation Criteria

The tester Verifies that there is a choice for VFC stock as she is VFC Eligible.
